### 그래프 만들기 방법

그래프라는 개념을 이해해보도록 하겠습니다. 이전에 우리는 집의 모든 친구들을 '그래프'라고 표현했죠. 이제는 그 '그래프'를 어떻게 만들어야 하는지 배우볼게요!

#### 필요한 정보: n과 m
먼저, 그래프를 만들기 위해 우리가 사용하는 코드는 두 개의 숫자가 필요합니다. 이 중 가장 중요한 두 수는 `n`과 `m`입니다.

- **n**: 이것은 그래프에서 모든 노드(위치)의 수입니다.
- **m**: 이것은 그래프에서 연결된 친구(인접한 노드)들의 수입니다.

우리가 알고 있는 예시로, n이 5이고 m이 6일 때 (그대로 적어도 하겠습니다), 그래프를 만들면 이렇게 될 것입니다:

```
[5] [n]
|   |
5   4
|   | 
3   2
|   | 
1   0
       | 
       6   m
```

이게 바로 '그래프'다! 각 노드는 다른 그래프에서의 친구를 나타내고, 각 노드와 연결된 친구는 인접한 노드라고 부릅니다.

#### 그래프 만들기: n과 m을 입력받아서

그렇다면 이렇게 많은 정보를 받아서 그래프를 만드는 방법이 있을까요? 아래에서 주목할 게 있습니다:

1. **n**을 `input()` 함수로 받습니다.
2. **m**을 `input()` 함수로 받습니다.

```python
# n과 m을 입력받아 그래프 만들기 예제
import sys

def create_graph(n, m):
    graph = [[0] * n for _ in range(n)]  # 모든 노드 연결 상태 초기화
    
    # 인접 행렬 생성: 노드와 친구 설정
    for i in range(m):
        x, y = map(int, input().split())  # 입력 받은 두 노드의 번호를 추출합니다.
        graph[x-1][y-1] = 1  # 친구 관계 저장
    
    return graph

# 사용자 입력 받기
n = int(input())
m = int(input())

graph = create_graph(n, m)

print(graph)
```

이 코드는 `input()` 함수를 통해 n과 m을 받아서 그래프를 만들며, 각 노드와 연결된 친구를 인접 행렬로 저장합니다.

#### 핵심 개념의 비유

그럼요, 이렇게 만들어진 '그래프'는 어떻게 우리에게 도움이 될까요? 

- **n**이 큰 경우: 우리 가족이나 그룹의 구성원들이 많을 때 그래프를 이용하면 누구와 어떤 관계에 있는지 쉽게 파악할 수 있습니다. 예를 들어, 5명 이상的朋友 network를 파악하려면 n가 크다는 것이죠.
  
- **m**이 많은 경우: 우리는 여러 사람이 연결되어 있는 지점들(인접한 노드)을 찾고 싶어 할 때 그래프를 이용하면 훨씬 쉽게 찾을 수 있습니다.

#### 예시 코드와 출력

```python
n = 5
m = 6

graph = create_graph(n, m)

print(graph)
```

이렇게 만들어진 그래프의 출력은 다음과 같아요:

```
[0] [1]
|   |
1   2
|   | 
3   4
|   | 
5   0  
       |   
       6   7
```

그런데 이 과정에서 우리는 숫자만 받아서 그래프를 만드는 것이 아니라, 실제 사람이 있는 사람들 사이의 관계를 표현하는 역할을 합니다. 예를 들어, n이 10이고 m이 3인 경우에 그래프는 다음과 같은 형태로 생겨납니다:

```
[10] [n]
|   |
9   8
|   | 
7   6
|   | 
5   4
|   | 
3   2
|   | 
1   0
       |   
       m
```

이제 n과 m을 입력받아 그래프를 만드는 방법을 알게 된 후, 당신의 프로젝트에 이런 형태의 정보를 사용하여 그래프를 만들려고 할 때 이 방법을 통해 도움을 받을 수 있을 거예요!

이제 여러분도 그래프라는 개념을 이해하고 그에 대한 실제 문제 해결 능력을 향상시키셨다면, 앞으로 많은 프로젝트에서 이 방법을 활용해 보시길 권장드립니다!

### 1\. map 객체란?

  * Type : **iterator(반복자)** < 특징 > 리스트는 데이터를 실제로 저장하지만 map은 "어떻게 바꿔줄지"의 **방법** 만 저장한다. 원본 데이터 (["1", "2", "3"])이 있다고 치고 적용할 함수 (int)라고 치고 현재 읽은 위치 (포인터)는 iterable하게 그래서 map은 새로 만들 때마다:



새로운 포인터를 가진 새 객체가 생성됨 원본 데이터를 처음부터 다시 읽을 수 있음 하지만 한 번 사용한 map 객체는 포인터가 끝에 가있어서 재사용 불가

#### MAP이 하는 일

`1. Take 2 parameter` `1-1.첫 번째는 적용할 함수 1-2.두 번째는 반복 가능한 객체` `2. 반복 가능한 객체의 각 요소에 함수를 적용한다` `3. 결과를 map 객체로 반환합니다`
    
    
    numbers = []
    string_list = input().split()  # ["1", "2", "3"]
    for string_num in string_list:
        numbers.append(int(string_num))
    print(numbers)  # [1, 2, 3]
